# dTackle

dTackle is a lightweight tackle script with a basic anticheat and no dependencies, a heavily edited version of https://forum.cfx.re/t/release-tackle-system/95129

## Installation

Take the dTackle folder and drag it into your resources.

In server.cfg, ensure you start config.cfg before the rest of the resource.

```cfg
exec @dTackle/config.cfg
ensure dTackle
```
